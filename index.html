Futuristic Search Interface – AI-Powered Search Engine

An AI-driven search engine that delivers precise and relevant results by combining advanced language models, vector databases, and multiple search engines. Retail Data Analysis – SQL & Python

A comprehensive eCommerce data analytics project that automates CSV data ingestion into MySQL, performs SQL-based analysis, and visualizes insights with Python. Key features include dynamic table creation, advanced queries, and visualizations of sales trends, customer behavior, and top-performing products, enabling actionable business insights.


Customer Churn Analysis – Exploratory Data Analysis (EDA)

An in-depth EDA project on a telecom dataset to identify key factors driving customer churn. Using Python, Pandas, and visualization libraries, the project analyzes contract types, payment methods, tenure, and demographics to uncover patterns and generate actionable insights. Recommendations include promoting long-term contracts, alternative payment methods, and targeted retention strategies for high-risk customer segments.

A lightweight Python command-line tool to track income, expenses, and visualize financial activity. Data is stored in a CSV file and analyzed using Pandas and Matplotlib.
